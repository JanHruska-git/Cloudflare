# Co je Cloudflare? - Školní projekt
Tento projekt je jednoduchá, moderní webová prezentace představující platformu **Cloudflare**, její účel a klíčové služby.
## Obsah projektu
* `index.html` - Hlavní HTML struktura s textovým obsahem a diagramem fungování.
* `style.css` - Moderní tmavý design (glassmorphism, plynulé přechody a animace).
## Jak spustit
1. Ujistěte se, že jsou všechny soubory ve stejné složce.
2. Dvakrát klikněte na soubor `index.html` pro spuštění ve webovém prohlížeči (např. Chrome, Edge, Firefox).
## Použité technologie
* HTML5
* CSS3 (vlastní styly bez frameworků, Google Fonts Outfit)
